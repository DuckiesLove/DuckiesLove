Project Training Plan

This outline helps new contributors get up to speed with the Talk Kink repository.

  1. Set up a local development environment by cloning the repo and running setup.sh.
  2. Review the survey JSON templates and JavaScript modules under template-survey.json and js/.
  3. Make incremental changes in feature branches and confirm that npm test passes before committing.
  4. Document updates in the README and verify pages locally using a simple web server.
  5. Open a pull request so changes can be reviewed and deployed to GitHub Pages.
